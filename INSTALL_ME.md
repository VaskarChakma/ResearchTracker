# STEP-BY-STEP INSTALLATION GUIDE

## Follow these exact steps:

### Step 1: Download the Extension
- Download the `ResearchTracker-main.zip` file
- Save it to your Desktop or Downloads folder

### Step 2: Extract the ZIP
**Windows:**
- Right-click on `ResearchTracker-main.zip`
- Click "Extract All..."
- Click "Extract"
- A folder named `ResearchTracker-main` will be created

**Mac:**
- Double-click `ResearchTracker-main.zip`
- It will automatically extract to a folder

### Step 3: Verify the Extraction
Open the `ResearchTracker-main` folder and make sure you see these files:
```
✓ manifest.json
✓ background.js
✓ content.js
✓ content.css
✓ popup.html
✓ popup.js
✓ popup.css
✓ dashboard.html
✓ dashboard.js
✓ dashboard.css
✓ icons (folder)
```

### Step 4: Open Chrome Extensions
1. Open Google Chrome
2. Type this EXACTLY in the address bar: `chrome://extensions/`
3. Press Enter

### Step 5: Enable Developer Mode
- Look at the TOP RIGHT corner of the page
- You'll see a toggle switch that says "Developer mode"
- Click it to turn it ON (it should turn blue)

### Step 6: Load the Extension
1. You should now see three buttons appear: "Load unpacked", "Pack extension", "Update"
2. Click the "Load unpacked" button
3. A file browser window will open

### Step 7: Select the Correct Folder
**This is the most important step!**

### Step 8: Verify Installation
If successful, you should see:
- "Research Paper Tracker" appear in your extensions list
- A card showing:
  - Extension icon
  - Name: "Research Paper Tracker"
  - Version: 1.0.0
  - Description: "Track and organize research papers"
  - No errors in red text

### Step 9: Pin the Extension
1. Look for the puzzle piece icon in Chrome's toolbar (top right)
2. Click it
3. Find "Research Paper Tracker"
4. Click the pin icon next to it
5. The extension icon will now appear in your toolbar

### Step 10: Test It
1. Click the extension icon
2. You should see a popup with statistics
3. Visit https://scholar.google.com
4. Search for any paper
5. You should see a "Save Paper" button appear
